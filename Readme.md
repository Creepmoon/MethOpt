## Краткое описание алгоритма решения ЗЛП (симплекс-метод)

- Задача линейного программирования (ЗЛП) приводится к каноническому виду: максимум целевой функции при линейных ограничениях вида ≤ и неотрицательных переменных.
- Добавляются переменные-слэки для ограничений ≤, формируется начальный опорный (базисный) план.
- Итерационно улучшается значение целевой функции, пока есть положительные редуцированные стоимости (для задачи на максимум).

Псевдокод (симплекс-метод, кратко):

```text
Вход: max c^T x, Ax ≤ b, x ≥ 0
1) Привести к каноническому виду: Ax + I s = b, s ≥ 0
2) Базис B ← столбцы I (свободные переменные — s), небазис N ← x
3) Повторять:
   3.1) Вычислить редуцированные стоимости: \bar{c}_N = c_N - c_B^T B^{-1} N
        - Если все \bar{c}_N ≤ 0 → оптимум достигнут, восстановить x*, s* и завершить.
   3.2) Выбрать входящую переменную j ∈ N с максимальной \bar{c}_j > 0
   3.3) Направление d_B = -B^{-1} N_{:,j}, d_j = 1, d_{N\{j}} = 0
   3.4) Правило минимального отношения для выхода из базиса:
        θ* = min_{i: (d_B)_i < 0} (x_B)_i / (-(d_B)_i)
        - Если подходящих i нет → задача неограничена (нет оптимума).
   3.5) Обновить базис: заменить вышедшую базисную переменную входящей, пересчитать B^{-1}, x_B
Пока
Выход: оптимальное решение (x*, s*), значение цели z* = c^T x*
```

Примечание: если нет очевидной начальной базы (например, есть ограничения вида ≥ или =), используется двухфазный метод с искусственными переменными: на Фазе I минимизируется сумма искусственных, на Фазе II решается исходная задача, стартуя из найденной базы.

## Рефлексия

- Полезные результаты:
  - Закрепил преобразование ЛП к стандартной форме (слэки, избыточные, искусственные).
  - Разобрался, как вычислять редуцированные стоимости через решение системы B^T y = c_B.
  - Отработал двухфазный симплекс: диагностика недопустимости через сумму искусственных.
- Трудности:
  - Корректная поддержка свободных переменных и восстановление исходных значений после решения.
  - Стабильное построение базиса при удалении искусственных переменных между фазами.
  - Численные нюансы: выбор порогов для сравнения с нулём, защита от вырожденности.
- Как преодолевались:
  - Ввёл разбиение свободных переменных на pos/neg и аккуратно собрал обратное отображение к исходным.
  - Реализовал явное решение B^T y = c_B и аккуратный пивот с нормировкой пивот-строки.
